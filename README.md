# 🚦 Sistema de Semáforo Inteligente

Projeto desenvolvido para a disciplina de **Matemática Computacional Aplicada**.

[![Abrir Simulação](https://img.shields.io/badge/🚦%20ABRIR%20SIMULAÇÃO-ONLINE-brightgreen?style=for-the-badge)](https://pedrodiogo7.github.io/A3-Matematica-Computacional-/)

## 🎯 Objetivo

Desenvolver um modelo computacional simplificado de um sistema de controle de semáforo inteligente utilizando:

- Sistemas numéricos binários
- Lógica proposicional
- Álgebra booleana
- Teoria dos conjuntos
- Tabela-verdade
- Máquina de estados
- Programação em Python

## 💻 Tecnologias

- Python
- HTML
- CSS
- JavaScript

## 🚦 Funcionalidades

- Simulação de semáforo inteligente
- Tabela-verdade interativa
- Máquina de estados
- Representação binária
- Interface visual para demonstração
